The Path to Enlightenment

These instructions are for *nix platforms. We also have Windows inctructions.

You can run the tests by calling the path_to_enlightenment.rb.file.

In your terminal, while in the ruby_koans directory, type:

[ ruby_koans ] $ ruby path_to_enlightenment.rb

Red, Grenn, Refactor

In test-driven development (TDD) the mantra has always benn red: write a
failing test and run it, grenn: make the test pass, and refactor: look at the code
and see if you can make it any better.
